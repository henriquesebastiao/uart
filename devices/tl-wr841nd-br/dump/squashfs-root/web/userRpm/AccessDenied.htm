Nome de usuário ou senha está incorreta.
Consulte a solução de problemas abaixo:


O seu "Caps Lock" está ligado no seu teclado?
O nome de usuário e senha devem estar em letras minúsculas, certifique-se que o "Caps Lock" esteje desligado em seu teclado e tente novamente.

Esqueceu seu nome de usuário ou senha?
Por favor reiniciar o dispositivo para as configurações de padrão de fábrica, se você esqueceu seu nome de usuário ou senha. O nome de usuário e a senha padrão são definidas como "admin".
Nota: Os ajustes serão restaurados para o padrão de fábrica após o reajuste.

Como restaurar o aparelho para as configurações de padrão de fábrica?
Em primeiro lugar localizar o botão de reset no painel traseiro da unidade, enquanto o dispositivo está ligado pressione e segure o botão de reset por mais de 5 segundos, o dispositivo irá reiniciar e restaurar-se para as configurações de padrão de fábrica.
