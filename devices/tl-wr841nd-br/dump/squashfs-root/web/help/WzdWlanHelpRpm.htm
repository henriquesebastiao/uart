Ajuda para Wireless

Interface de Rádio - Ativar ou desativar o wireless rádio.

Nome da Rede Wireless - Insira uma sequência de até 32 caracteres. O mesmo Nome (SSID) deve ser atribuído a todos os dispositivos wireless em sua rede. O SSID padrão está configurado para ser _xxxxxx (xxxxxx indica os únicos últimos 6 caracteres do MAC Adress de cada Roteador), que pode garantir a segurança da sua rede wireless. Mas é altamente recomendável que você altere o nome da sua rede (SSID) para um valor diferente. Este valor é sensível. Por exemplo, MYSSID NÃO é o mesmo que MySsid.

Região - Selecione sua região da lista pull-down. Este campo especifica a região onde a função wireless do Roteador pode ser usada. Pode ser ilegal usar a função wireless em outra região que não seja uma das especificadas neste arquivo. Se o seu país ou região não está listada, por favor, entre em contato com a agência governamental local para assistência.

Modo - Este campo determina em qual modo wireless o Roteador trabalha.

Canal - Este campo determina qual a frequência de operação será usada. Não é necessário alterar o canal wireless, a menos que você perceba problemas de interferência com outro ponto de acesso nas proximidades. Se você selecionar auto, então o AP selecionará o melhor canal automaticamente.

Largura do Canal - A largura de banda do canal wireless.

Você pode selecionar uma das seguintes opções de segurança:

Desabilitar a Segurança - A função de segurança wireless pode ser ativada ou desativada. Se desativada, as estações wireless estarão aptas a conectar o Roteador sem criptografia. É altamente recomendável que voce escolha uma das opções abaixo para ativar a segurança.

WPA-PSK/WPA2-PSK - Selecione WPA baseada em senha pré-compartilhada.

  • Senha Wireless - Você pode inserir caracteres ASCII ou Hexadecimais.
  • Para ASCII, o comprimento deve ser entre 8 e 63 caracteres.
  • Para Hexadecimal, o comprimento deve ser entre 8 e 64 caracteres.
    • Por favor, note que as senhas são sensíveis.

Não Alterar - Se você escolher esta opção, a configuração de segurança não será alterada!
