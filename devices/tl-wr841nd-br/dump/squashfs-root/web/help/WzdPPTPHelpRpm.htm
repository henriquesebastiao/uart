Ajuda do PTTP

Nome de Usuário e Senha - Insira o Nome de Usuário e Senha fornecidos por seu Provedor. Estes campos são case-sensitive.

Selecione IP Estático se o endereço IP, máscara de sub-rede, gateway padrão e servidor DNS foram fornecidos por seu Provedor. Caso contrário, por favor, selecione IP Dinâmico.

Endereço IP/Nome do Servidor - Insira o endereço IP ou nome de domínio fornecido por seu Provedor.

Clique em Próximo para continuar, ou clique em Retornar para voltar a página anterior.
