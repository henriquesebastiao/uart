Ajuda sobre Wireless - Avançado

  • Potência de Transmissão - É possível especificar a potência de transmissão de saída do seu roteador. Pode-se selecionar entre Alto, Médio ou Baixo. A configuração padrão e recomendada é a opção Alto.
  • Intervalo Beacon - Os beacons são os pacotes enviados pelo Roteador para sincronizar a rede sem fios. O valor do Intervalo Beacon determina o intervalo de tempo de envio destes pacotes. Você pode especificar um valor entre 40 a 1000 milissegundos. O valor padrão é 100.
  • Limite de RTS - Você pode especificar o Limite de RTS (Pedido de Transmissão). Se o pacote é maior que o tamanho do Limite de RTS especificado, o Roteador irá enviar os quadros RTS para uma estação particular de recebimento e negociar a transmissão de um quadro de dados. O valor padrão é 2346.
  • Início de Fragmentação - Este valor é o tamanho máximo determinante se os pacotes serão fragmentados. Se definir o limite de fragmentação muito baixo pode resultar em mau desempenho da rede desde que os pacotes se excedam. A configuração padrão é 2346 e é a recomendada. (Este valor é para o modo 11n não pode ser alterado)
  • Intervalo DTIM - Este valor determina o intervalo da Mensagem de Indicação para Entrega do Tráfego DTIM. Você pode especificar o valor entre Intervalos Beacon de 1 a 255. O valor padrão é 1, o que indica que o intervalo DTIM é igual ao Intervalo Beacon.
  • Habilitar WMM - A função WMM (Wireless Multimedia) pode garantir os pacotes com mensagens de prioridade alta sejam transmitidos preferencialmente. É recomendado que esta opção seja habilitada.
  • Habilitar Intervalo de Controle - Esta função é recomendada para o aumento da capacidade de transmissão dos dados, pois assegura que transmissões distintas não interfiram umas com as outras.
  • Habilitar Isolamento do Ponto de Acesso - Se habilitada, esta função irá isolar todas as estações sem fios conectadas ao roteador, para que estas não possam se comunicar entre si (aplicado se roteador for instalado em local público, como um aeroporto). Esta função será desabilitada se o WDS for habilitado.

Nota: Se não estiver familiarizado com os parâmetros de configuração desta página, é recomendado manter os valores padrões fornecidos. Caso contrário as alterações poderão resultar em desempenho inferior da rede sem fios.
