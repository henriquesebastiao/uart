Ajuda para Criar uma Entrada de Estaçõe

Você poderá criar uma entrada para a lista de Estações nesta página.

  • Tipo de Endereço - Selecione entre as opções IP ou MAC.
  • Descrição do Estaçõe - Neste campo, criar uma descrição única para o estaçõe a ser adicionado.

Se a opção IP for selecionada, você verá o seguinte campo:

  • Endereço(s) IP (LAN) - Digite o endereço ou uma série de endereços IP dos estações que deseja adicionar. Exemplo: 192.168.0.23.

Se a opção MAC for selecionada, você verá o seguinte campo:

  • Endereço MAC - Digite o endereço MAC do estaçõe no formato XX-XX-XX-XX-XX-XX. Exemplo: 00-11-22-33-44-AA.

Clique no botão Avançar para continuar, ou no botão Voltar para voltar à página anterior.
