Ajuda sobre Controle de Acesso - Adicionar ou Modificar Alvos

Você pode criar uma entrada para a lista de alvo nesta página.

  • Modo - Selecione entre as opções Endereço IP e o Nome de Domínio.
  • Descrição do Alvo - Digite um nome específico para cada alvo a ser criado.

Se a opção Endereço IP for selecionada, você verá os seguintes itens:

  • Endereço(s) IP - Digite o endereço IP ou série de endereços dos alvos escolhidos para serem adicionados à entrada.
  • Porta(s) - Especifique a porta ou a série de portas para o alvo em questão.
  • Protocolo - Selecione entre as opções Todos, TCP, UDP, e ICMP.
  • Portas de Serviços Mais Comuns - listagem de portas de serviços comuns. Selecione uma das opções na lista suspensa, e o número da porta correspondente será preenchido automaticamente.

Se a opção Nome de Domínio for selecionada, você verá os seguintes itens:

  • Domínios - Aqui você pode entrar com até 4 nomes de domínio, podendo ser a URL completa (www.google.com.br) ou palavras chave (google) - neste segundo exemplo, qualquer endereço de internet que contenha a palavra 'google' (www.google.com.br, www.google.cn...) será bloqueado ou permitido.

Clique em Salvar para armazenar as configurações.

Clique em Voltar para retornar à página anterior.
