Ajuda sobre Controle de Acesso - Adicionar ou Modificar Estações

Você poderá criar uma entrada para a lista de Estações nesta página.

  • Tipo de Endereço - Selecione entre as opções IP ou MAC.
  • Descrição da estação - Neste campo, criar uma descrição única para o estaçõe a ser adicionado.

Se a opção IP for selecionada, você verá o seguinte campo:

  • Endereço(s) IP (LAN) - Digite o endereço ou uma série de endereços IP das estações que deseja adicionar. Exemplo: 192.168.0.23.

Se a opção MAC for selecionada, você verá o seguinte campo:

  • Endereço MAC - Digite o endereço MAC da estação no formato XX-XX-XX-XX-XX-XX. Exemplo: 00-11-22-33-44-AA.

Clique em Salvar para armazenar as configurações.

Clique em Voltar para retornar à página das Configurações das Estações.
