Ajuda para Crie uma Entrada Avançada de Horários

Nesta página é possível criar uma entrada para um período de tempo que envolve dias e horas, que poderá ser aplicada em uma regra de controle de acesso à internet.

  • Descrição - Defina uma descrição específica para cada entrada criada.
  • Dias - Selecione entre as opções Todos os dias e Selecionar dias.
  • Período - Selecione a opção O dia todo (24 horas), ou desmarque-a para inserir o Período Inicial e Final desejados.
  • Período Inicial - Digite o tempo inicial no formato HHMM (HHMM são 4 números). Por exemplo, 0800 é igual a 8:00.
  • Período Final - Digite o tempo final no formato HHMM (HHMM são 4 números). Por exemplo, 2000 é igual a 20:00.

Clique no botão Avançar para continuar, ou no botão Voltar para voltar à página anterior.
