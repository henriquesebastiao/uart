Ajuda Estatísticas wireless

Esta página apresenta Endereço MAC, Status Atual, Pacotes recebidos e Pacotes enviados para cada estação wireless conectada.

  • Endereço MAC - Endereço MAC da estação wireless conectada.
  • Status Atual - estado de funcionamento da estação wireless conectada, uma das STA-AUTH / STA-ASSOC / STA-JOINED / WPA / WPA-PSK / WPA2 / WPA2-PSK / AP-UP / AP-DOWN / Desconectada.
  • Pacotes recebidos - pacotes recebidos pela estação.
  • Pacotes enviados -pacotes enviados pela estação.
  • Configurar - Botão utilizado para carregar o item à lista Filtrar MAC.
    • Negar - Caso a função Filtrar MAC esteja habilitada, nega o acesso à estação.
    • Permitir - Caso a função Filtrar MAC esteja habilitada, permite o acesso à estação.

Para atualizar essa página e para mostrar as atuais estações wireless conectadas, clique no botão Atualizar .

Se o número de estações wireless conectadas ultrapassam uma página, clique no botão Avançar para ir para a próxima página ou clique no botão Voltar para retornar à página anterior.

Nota: Esta página será atualizada automaticamente a cada 5 segundos.
