Ajuda para Criar uma Entrada de Controle de Acesso à Internet

Você pode criar uma nova regra de controle de acesso nesta página.

  • Nome da Regra - Definir um nome para a regra. Note que este nome deve ser exclusivo para cada regra criada.
  • Estaçõe - Selecione uma entrada de computadores previamente cadastrados para adicionar à nova regra.
  • Alvo - Selecione um alvo da lista suspensa para adicionar à regra. O valor padrão é Qualquer Alvo.
  • Horário - Selecione um horário da lista suspensa para adicionar à regra. O valor padrão é Qualquer horário.
  • Estado - Selecione Habilitado para ativar a regra no sistema, ou Desabilitado para que a mesma seja desativada.

Clique no botão Finalizar para aplicar a nova Política de Controle de Acesso, ou no botão Voltar para voltar à página anterior.
