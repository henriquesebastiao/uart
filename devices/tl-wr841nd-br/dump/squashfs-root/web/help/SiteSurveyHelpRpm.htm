Ajuda sobre WDS - Lista de pontos de acesso

Nota: As informações dos pontos de acesso que você poderá conectar são exibidas nesta página. Siga as orientações para estabelecer uma conexão:

  • Selecione a rede sem fios na qual deseja se conectar através deste roteador para iniciar a operação WDS;
  • Clique em "Conectar";
  • O SSID da rede de destino será preenchido no lugar correto da página de configuração WDS automaticamente.

Aperte o botão Atualizar para renovar a lista de pontos de acesso disponíveis.

Aperte o botão Voltar para retornar à página de configuração da interface Wireless.
