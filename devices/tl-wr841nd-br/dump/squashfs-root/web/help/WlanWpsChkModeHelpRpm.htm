Ajuda sobre WPS/QSS - Adicionar novo Computador

Este guia irá orientá-lo para adicionar um novo equipamento wireless para uma rede existente manualmente. Se o novo equipamento suportar o WPS e for equipado com um botão de configuração, você pode adicioná-lo para a rede apertando o botão de configuração no equipamento, despois escolha o botão "Aperte o botão WPS do novo computador dentro de dois minutos" e clique o botão Conectar nesta página web de configuração dentro de dois minutos. Além disso, você pode adicionar também o novo equipamento entrando o PIN deste equipamento e depois clique o botão Conectar.

  • Digite o número PIN do novo computador - Escolha esta configuração se o novo equipamento suportar o caminho da conexão entrando seu PIN.
  • Aperte o botão WPS do novo computador dentro de dois minutos - Escolha esta opção se o novo computador suportar o protocolo WPS através de um botão.
  • Voltar - Retornar para a página de configuração.
  • Conectar - Conectar o novo computador à rede existente.
