Ajuda sobre Firewall - Lista de computadores bloqueados

Esta página exibe as informações de Endereço IP e Endereço MAC dos computadores que foram bloqueados pelo protocolo DoS (Denial of Service) do Roteador.

Para atualizar esta página e exibir a atual lista de computadores bloqueados, clique no botão Atualizar.

Clique no botão Limpar para limpar todas as entradas exibidas, e assim liberar os computadores bloqueados.

Clique no botão Voltar para retornar à página anterior.
