Ajuda sobre Interface WAN

Se você não souber como escolher o tipo de conexão apropriada, clique no botão Detectar para permitir que o Roteador identifique automaticamente a conexão de Internet que você utiliza. O tipo de conexão será relatado quando um serviço ativo de Internet é detectado com sucesso pelo Roteador. Para ter certeza que o tipo de conexão fornecido pelo seu Provedor de Internet, por favor entre em contato com seu Provedor de Internet.

Referente aos parâmetros de configuração utilizados para a opção PPPoE:

Nome de Usuário/Senha - Entre o Nome de Usuário e a Senha fornecidos pelo seu Provedor de Internet. Estes campos são sensíveis a maiúsculas e minúsculas.

Conexão Secundária - Disponível apenas para conexão PPPoE. Caso seu Provedor de Internet fornecer um tipo de Conexão extra tal como IP Dinâmico/Estático para conectar-se à uma rede local, depois você pode marcar a opção desejada (IP Dinâmico/Estático) para ativar esta conexão secundária.

  • Desabilitado - A conexão secundária é desabilitada por padrão, portanto existe apenas uma conexão PPPoE. Isso é recomendado.
  • IP Dinâmico - Use o endereço IP dinâmico para conectar-se à rede local fornecida pelo Provedor de Internet.
  • IP Estático - Use endereço do IP estático para conectar-se à rede local fornecida pelo Provedor de Internet.
    • Endereço IP - Insira o endereço IP fornecido pelo Provedor de Internet para a conexão secundária. Este endereço é usado apenas para accessar a rede local da conexão secundária.
    • Máscara de Sub-rede - Insira a máscara de sub-rede fornecida pelo Provedor de Internet para a conexão secundária.

Conexão sob demanda - Você pode configurar o Roteador para desconectar sua conexão a Internet depois de um período especificado da conectividade a Internet (Tempo Máximo de Ociosidade). Se a sua conexão de Internet encerrar-se em função da inatividade, a Conexão sob demanda permite que o Roteador restabeleça automaticamente sua conexão quando você tentar acessar a Internet novamente. Se você desejar ativar a Conexão sob demanda, selecione esta opção. Se você quiser que sua conexão Internet fique ativa o tempo todo, digite 0 no campo Tempo Máximo de Ociosidade.

Nota: Às vezes a conexão não pode ser desconectada embora você especifique um tempo para o Tempo Máximo de Ociosidade(0~99 minutos) porque algumas aplicações visitam a Internet continuamente no contexto.

Conectar Automaticamente - Conectar automaticamente depois que o Roteador se desconectar. Para usar esta opção, selecione esta opção.

Conexão baseada em tempo - Você pode configurar o Roteador para que ele seja conectado ou desconectado com base no tempo. Entre o tempo inicial em HH-MM para conexão e tempo final em HH-MM para desconexão nos campos Período de Tempo .

Conectar Manualmente - Você pode configurar o Roteador para que seja conectado ou desconectado manualmente. Depois de um período especificado de inatividade (Tempo Máximo de Ociosidade),o Roteador irá desconectar sua conexão a Internet, e não poderá restabelecer sua conexão automaticamente quando você tentar acessar a Internet novamente. Se você quiser que sua conexão Internet fique ativa o tempo todo, entre 0 no campo Tempo Máximo de Ociosidade. Senão, entre o número em minutos que você deseja ter para a conexão Internet a menos que seja pedido um novo link.

Nota: Às vezes a conexão não pode ser desconectada embora você especifique um Tempo Máximo de Ociosidade (0~99 minutos) porque algumas aplicações visitam a Internet continuamente no contexto.

Nota: Apenas quando você estabelece o tempo do sistema na página Ferramentas do Sistema -> Relógio, a função Conexão baseada no tempo pode fazer efeito.

Clique no botão Conectar para conectar imediatamente.

Clique no botão Desconectar para desconectar imediatamente.

Clique no botão Avançado para estabelecer as opções avançadas.

Clique no botão Salvar para salvar suas configurações.
