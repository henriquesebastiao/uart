Ajuda sobre Interface WAN

Se você não souber como escolher o tipo de conexão apropriada, clique no botão Detectar para permitir que o Roteador identifique automaticamente a conexão de Internet que você utiliza. O tipo de conexão será relatado quando um serviço ativo de Internet é detectado com sucesso pelo Roteador. Para ter certeza que o tipo de conexão fornecido pelo seu Provedor de Internet, por favor entre em contato com seu Provedor de Internet.

Referente aos parâmetros de configuração utilizados para a opção L2TP:

Nome de Usuário/Senha - Entre o Nome de Usuário e a Senha fornecidos pelo seu Provedor de Internet. Estes campos são sensíveis a maiúsculas e minúsculas.

IP Dinâmico/IP Estático - Selecione IP Estático se o endereço IP, a máscara de sub-rede, o gateway e o endereço do servidor DNS foram fornecidos pelo seu Provedor de Internet. Senão, por favor selecione IP Dinâmico.

Endereço IP do Servidor/Nome - Entre o endereço IP do servidor ou o nome de domínio fornecido pelo seu Provedor de Internet.

Endereço IP - Entre o endereço IP usado para discar. (Pode ser configurado apenas quando o IP Estático é selecionado).

Máscara de Sub-rede - Entre a máscara de sub-rede fornecida pelo seu Provedor de Internet. (Pode ser configurada apenas quando o IP Estático é selecionado).

Gateway - Entre o gateway fornecido pelo seu Provedor de Internet. (Pode ser configurado apenas quando o IP Estático é selecionado).

DNS - Entre o servidor DNS fornecido pelo seu Provedor de Internet. (Pode ser configurado apenas quando o IP Estático é selecionado).

Endereço IP na Internet O endereço IP na Internet atribuído pelo servidor L2TP.

DNS da Internet O endereço do servidor DNS na Internet atribuído pelo servidor L2TP .

Conexão sob demanda - Você pode configurar o Roteador para desconectar a sua conexão Internet depois de um período especificado da conectividade à Internet (Tempo Máximo de Ociosidade). Se a sua conexão à Internet encerrar-se em função da inatividade, a Conexão sob demanda permite que o Roteador restabeleça automaticamente sua conexão enquanto você tenta acessar a Internet novamente. Se você desejar ativar a Conexão sob demanda, selecione esta opção. Se você quiser que a sua conexão à Internet fique ativa o tempo todo, entre 0 no campo Tempo Máximo de Ociosidade. Senão, entre o número de minutos que você quer ter passado antes de sua conexão à Internet terminar.

Nota - Às vezes a conexão não pode ser desconectada embora você especifique um momento para o Tempo Máximo de Ociosidade porque algumas aplicações visitam a Internet continuamente no contexto.

Conectar Automaticamente - Conectar automaticamente depois da desconexão do Roteador. Para usar esta opção, clique no botão rádio.

Conectar Manualmente - Você pode configurar o Roteador para que ele conecte ou desconecte manualmente. Depois de um período especificado de inatividade (Tempo Máximo de Ociosidade),o Roteador irá desconectar sua conexão Internet, e não poderá restabelecer sua conexão automaticamente enquanto você tenta acessar a Internet novamente. Para usar esta opção, selecione-a. Se você quiser sua conexão à Internet para ficar ativa o tempo todo,entre 0 no campo Tempo Máximo de Ociosidade . Senão, entre o número em minutos que você deseja ter como duração da conexão à Internet a menos que seja solicitado um novo link.

Nota - Às vezes a conexão não pode ser desconectada embora você especifique um tempo para o Tempo Máximo de Ociosidade porque algumas aplicações visitam a Internet continuamente no contexto.

Clique no botão Conectar para conectar imediatamente.

Clique no botão Desconectar para desconectar imediatamente.

Clique no botão Salvar para armazenar suas configurações.
