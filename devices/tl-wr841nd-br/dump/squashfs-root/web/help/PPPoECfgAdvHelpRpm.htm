Ajuda sobre Interface WAN - Configurações Avançadas PPPoE

Tamanho MTU - O tamanho padrão MTU(Unide Máxima de Transmissão) é de 1480 bytes, o que geralmente é bom. Para alguns Provedores de Internet, você precisa modificar o MTU. Isso não deverá ser feito a menos que tenha certeza que é necessário para o seu Provedor de Internet.

Nome do Serviço/Nome do CA - O Nome do Serviço e o Nome do CA (Concentrador de Acesso) , não deverá ser feito a menos que tenha certeza que é necessário para o seu Provedor de Internet.

Endereço IP especificado pelo Provedor de Internet - Se sabe se seu Provedor de Internet não transmite automaticamente o endereço IP para o Roteador durante a conexão, clique na caixa de botão "Usar Endereço IP especificado pelo Provedor" e insira o endereço IP fornecido pelo Provedor de Internet, em notação decimal e separado por pontos.

Detectar Intervalo Online - o valor padrão é 0, você pode entrar o valor entre 0 e 120. O Roteador irá detectar o Concentrador de Acesso online a cada intervalo de segundos. Se o valor é 0, significa que não há detecção.

Usar os seguintes servidores DNS - Se o seu Provedor de Internet especificar um endereço IP do servidor DNS para você, clique na caixa de seleção, e preencha os espaços vazios abaixo DNS Primário e DNS Secundário . O DNS Secundário é opcional.Senão, os servidores DNS serão atribuídos dinamicamente do Provedor de Internet.

DNS Primário - Entre com o endereço IP do DNS fornecido pelo seu Provedor de Internet em notação decimal e separados por pontos.

DNS Secundário - (Opcional) Entre com o segundo endereço IP do DNS fornecido pelo seu Provedor de Internet em notação decimal e separados por pontos.

Nota: Os novos parâmetros avançados PPPoE não farão efeito até a sua discagem de novo.

Clique no botão Salvar para armazenar suas configurações.

Clique no botão Voltar para retornar a página anterior.
