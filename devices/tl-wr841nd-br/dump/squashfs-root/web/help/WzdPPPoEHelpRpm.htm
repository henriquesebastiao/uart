Ajuda do PPPoE

Nome de Usuário e Senha - Insira o Nome de Usuário e Senha fornecidos por seu Provedor. Estes campos são case-sensitive.

Conexão Secundária - Conexão Secundária - Está disponível apenas para conexão PPPoE. Se seu provedor fornece uma conexão extra como IP Dinâmico ou Estático para conectar a uma rede local, você pode marcar o botão IP Dinâmico ou Estático para ativar esta conexão secundária.

  • Desabilitado - A conexão secundária é desabilitada por padrão, portanto somente haverá a conexão PPPoE (Recomendável).
  • IP Dinâmico - Utilize endereço de IP dinâmico para conectar à rede fornecida por seu Provedor.
  • IP Estático - Utilize endereço de IP estático para conectar a rede fornecida por seu Provedor. Endereço IP - Endereço IP - Insira o endereço IP fornecido por seu Provedor para conexão secundária. Este endereço é utilizado somente para acessar a rede local da conexão secundária. Máscara de Sub-rede - Insira a máscara de sub-rede fornecida por seu Provedor para conexão secundária.
