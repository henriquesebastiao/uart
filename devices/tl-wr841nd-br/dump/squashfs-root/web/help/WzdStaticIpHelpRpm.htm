Ajuda sobre IP Estático

Os parâmetros de IP devem ser fornecidos pelo seu Provedor de Internet.

  • Endereço IP - Este é o endereço IP da WAN como visto por usuários externos da Internet (inclusive seu Provedor de Internet). Entre o endereço IP no campo.
  • Máscara de Sub-rede - Digite a Máscara de Sub-rede a ser usada para o endereço IP da WAN.
  • Gateway Padrão - Digite o gateway padrão no espaço designado.
  • DNS Primário - Digite o endereço IP do servidor DNS.
  • DNS Secundário - Se o seu Provedor de Internet fornecer um outro endereço IP do DNS, digite-o neste campo.

Clique no botão Avançar para continuar, ou no botão Voltar para retornar à página anterior.
