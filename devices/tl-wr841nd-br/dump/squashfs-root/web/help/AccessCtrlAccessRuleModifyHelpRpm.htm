Modificar Ajuda de Controle de Entrada de Internet

Você pode modificar a regra nesta página.

  • Nome da Regra - Neste campo, modifique um nome para a regra. Note que este nome deve ser único.
  • Parte Estaçõe - Nesta parte você pode mudar a Descrição do Estaçõe, Tipo de Endereço ou Endereço(s) IP (LAN). Note que irá substituir a antiga Regra de Host.
  • Parte Alvo - Nesta parte você pode mudar qualquer informação sobre a Parte Alvo. Note que isso vai substituir a antiga Regra de Alvo.
  • Parte Horário - Nesta parte, você pode também pode alterar qualquer informação sobre a Parte Agenda. Note que isso vai substituir a antiga Regra de Agenda.
  • Estado - Neste campo, existem duas opções, Habilitado ou Desabilitado. Selecione Habilitado para que a regra tenha efeito. Selecione Desabilitado e a regra não terá efeito.

Clique Salvar para completar as configurações.

Clique Voltar para retornar para a página de Gerenciamento de Regras de Controle de Acesso.
