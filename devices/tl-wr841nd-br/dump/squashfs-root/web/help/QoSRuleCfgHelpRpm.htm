Ajuda sobre Controle de Banda - Adicionar ou Modificar Regras

Esta página permite definir parâmetros de configuração para controle de banda por usuário e aplicação, permitindo de forma flexível determinar uma largura de banda limitada (que restringe o acesso) ou dedicada (que potencializa a conexão).

Para configurar os parâmetros de controle de banda, siga as orientações abaixo:

  • Habilitar - marque a opção para ativar as configurações.
  • Endereço(s) IP - Endereço ou Série de endereços IP dos computadores conectados na rede local. O controle de banda será realizado sobre os endereços determinados neste campo.
  • Porta(s) - Porta ou série de portas relacionadas às aplicações que irão ingressar o controle de banda da regra criada. Por exemplo, se é desejado limitar a velocidade de conexão para pacotes de vídeos, digitar a porta responsável pelo estabelecimento das conexões de acordo com o software em uso.
  • Protocolo(s) - escolha um protocolo da camada de transporte - Todos, TCP ou UDP.
  • Velocidade de Saída - Determine a velocidade máxima e mínima para o tráfego de upload do link de internet. Determinar apenas a velocidade máxima irá LIMITAR a velocidade de conexão; determinar a velocidade mínima irá DEDICAR a capacidade de transmissão para os computadores e aplicações definidos na regra.
  • Velocidade de Entrada - Determine a velocidade máxima e mínima para o tráfego de download do link de internet. A regra descrita acima também é válida para estes campos.

Clique no botão Salvar para armazenar as configurações.
