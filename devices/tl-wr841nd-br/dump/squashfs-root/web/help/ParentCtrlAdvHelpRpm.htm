Controle de Pais - Adicionar ou Modificar Regra

Nesta página é possível definir os parâmetros de controle de domínio para os computadores que deverão ter o acesso restrito à internet para determinadas páginas.

  • Endereço MAC computador Controlado - Digite o endereço físico (MAC) do computador que você deseja controlar. Se deixar este campo em branco, a regra será aplicada a todos os computadores da rede local, exceto para o computador do responsável pela gestão da rede.
  • Computadores conectados - É possível visualizar todos os computadores atualmente conectados na interface LAN do roteador. Escolha um deles para preencher o campo Endereço MAC computador Controlado.
  • Descrição dos Domínios - Neste campo, definir um nome para a regra de permissão a ser definida. Esta descrição deve ser diferente das demais regras que por ventura sejam criadas.
  • Domínios Permitidos - Neste campo você poderá definir até 8 páginas/domínios que serão permitidos para o acesso da estação controlada, sendo que as informações a serem preenchidas podem variar entre:
    •  
    • Nome completo ou palavra chave - cartoon (permitirá o acesso a todas as páginas que contiverem esta palavra).
    • URL (domínio) - www.cartoonnetwork.com.br (permitirá o acesso somente para esta URL completa.

  • Período - Neste campo, escolha o tempo efetivo para a utilização da regra (tempo em que ela ficará habilitada). Você pode também definir dias e horários manualmente, através das configurações de horário localizadas em Controle de Acesso > Horários. O valor padrão é Todo momento.
  • Estado - Selecione entre Habilitado ou Desabilitado para ativar ou não a regra de controle.

Clique em Salvar para armazenar as configurações.

Clique em Voltar para retornar à página da anterior.
